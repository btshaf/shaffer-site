# bshaffer.co

Personal portfolio site built with Next.js 15.

## Tech Stack
- Next.js 15 with App Router
- TypeScript
- Tailwind CSS
- Markdown content management
- Vercel deployment

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Content Management

Content is managed through Markdown files in the `/content` directory:
- `/content/site.md` - Hero and about sections
- `/content/experience/` - Work experience entries
- `/content/case-studies/` - Portfolio case studies

## Deployment

Deployed to [bshaffer.co](https://bshaffer.co) via Vercel with automatic builds on push to main.
